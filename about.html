Project Idea

This project is a static website for restaurant table booking. It helps users view table availability, choose pages easily, and complete login and signup actions.

The website also demonstrates JavaScript features required in the assignment, such as prompt dialogs, if...else decisions, switch selection, form validation, functions, events, repetition and changing HTML and CSS with script.
